Concept Description Example
Opening Tag This tag lets our browser know what version of HTML we are using. On this page, it is HTML 5. Other versions may use different keywords. Note that no end tag is needed. < !DOCTYPE html >
Basic HTML Tag After the tag indicating which version of HTML version, this signals to the browser that we are ready to incorporate elements into our document. An end tag is needed at the bottom of the document. < html >
Head Tag Contains other elements on a page and indicates general info about page. Like the HTML tag, it needs be closed after we indicate title info. < head >
Title Tag Title appears in browser's title bar or page tab; often used by search engines to rank websites. Also needs to be closed before head tag is. < title >
Body Tag Presents main content, including text, images, and links. Needs closing tag after content finished. < body >
UTF Tag Optional but allows for us to use and see a wide range of characters, numbers, and symbols. Does not need closing tag. < meta charset="UTF-8" >
Heading Tag Has six levels, with 1 being the largest (used for important headings, often the top of a page) and six for the smallest, like footnotes. Needs a closing tag. < h2 > Medium Heading < /h2 >
