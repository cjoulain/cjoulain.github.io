Enumerables

This week we learned more about enumerables. Enumerables frequently involve something that can be counted. Through this, you can sort, search, collect and more across a class of objects. What makes it effective is the fact that it can do this across a set. This makes it ideal for structures like arrays and hashes, which involve ordered groupings of things.

.Map is a handy function for iteration. A related function is .collect. Here is a simple example of .map in action:

words = [“this”, “that”, “here”]
words.map { |i| i.reverse}

This returns:

[“siht”, “taht”, “ereh”]

Unlike a function like each, it returns the entire set (not just one string). Keep in mind that .map is also ideal if you want to change the return value of your data. If you’re looking to preserve your original information, .each is the way to go.
